# twitter-trends-bot

A bot to post Google Trends on Twitter

# To run

1. Set your twitter credentials at _.credentials/twitter-credentials.json_

```js
{
    "apiKey": "",
    "apiSecretKey": "",
    "accessToken": "",
    "accessTokenSecret": ""
}
```

2. Run the bot with **YARN** or **NPM**

```js
    yarn install && yarn start
    npm install && npm start
```
